How it works

1. Audio File Upload

The system accepts audio files in WAV format for analysis. Users upload their WAV files, which are processed directly to ensure consistency in audio input.

Feature Extraction

Once the WAV file is uploaded, important features such as MFCCs, spectral roll-off, and chroma features are extracted using the Librosa library. These extracted features are then normalized to ensure consistency and accuracy.

Genre Prediction

The normalized features are then input into two pre-trained machine learning models, which predict the genre of the music. The system consolidates predictions from both models to deliver a final, accurate genre classification.
